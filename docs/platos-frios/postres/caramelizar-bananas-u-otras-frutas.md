# Caramelizar bananas u otras frutas

## Proporciones
Esto es para 2 bananas cortadas en rebanadas.

En una sartén colocar 3 cucharadas de manteca.

Una vez bien caliente agregar 3 cucharadas de azúcar (blanca o negra) y esperar que tome color caramelo.

Revolver poco con cuchara de madera.

## Para servirlas como postre
Antes de agregar las bananas, incluir el jugo de ½ limón y el jugo de ½ naranja, ron o canela.
