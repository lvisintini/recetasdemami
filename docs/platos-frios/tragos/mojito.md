# Mojito

Machacar:

- azúcar (una cucharadita)
- Menta (una ramita)

Agregar:
- ½ jugo de limón
- hielo bien picado
- ½ vaso de ron

Puede agregarse seven up.

Decorar con hojitas de menta.
